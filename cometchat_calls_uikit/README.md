## About the project
<a href="https://pub.dev/packages/cometchat_calls_uikit"><strong>CometChat Calls UI Kit</strong></a> provides pre-built user interface kit that developers can use to quickly integrate into  and create a reliable & fully featured chat and calls experience into an existing or a new mobile app.<br />

### Built With
- Flutter 3.10.5
- Dart 3.0.5
- Android Studio 2022.2
- Xcode 14.3
- Android 9, API 28 and above
- iOS 12.0

## Getting Started
To set up Flutter Calls UIKit and utilize CometChat for your chat and calls functionality, you'll need to follow these steps:
- Registration: Go to the CometChat website (https://www.cometchat.com/) and sign up for an account.
- Create an App: After registering, log in to your CometChat account and create a new app. Provide the necessary details such as the app name, platform (Flutter), and other relevant information. Once you've created the app, CometChat will generate an Auth Key and App ID for you. Make sure to keep these credentials secure, as you'll need them later.
- Make sure you've read the [key concepts](https://www.cometchat.com/docs/v4/flutter-uikit/key-concepts)
- Follow the steps provided in the integration section [here](https://www.cometchat.com/docs/v4/flutter-uikit/calls-integration)
